# Simple Notepad (Java Swing Project)

## Project Description
This project is a simple **Notepad application** built using **Java Swing** as part of the *IT3003: Advanced Programming Techniques* coursework.  
The application allows users to create, open, edit, and save text files, similar to a basic text editor.

---

## Features
- **File Menu**
  - New (clear the text area)
  - Open (load a text file)
  - Save (save text to a file)
  - Exit (close the application)

- **Edit Menu**
  - Cut
  - Copy
  - Paste

- **Help Menu**
  - About (displays developer name & ID)

- **Format Menu (Optional)**
  - Change Font Color
  - Change Background Color
